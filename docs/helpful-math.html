Helpful math¶

Probability and statistics are branches of mathematics.

Algebra¶

\(a^{-n} = \frac{1}{a^{n}}\) \(a^{m/n} = n \sqrt{a^{m}}\)
\(a^{1/n} =n \sqrt{a}\) \(a^{mn} = (a^{m})^{n}\)
\(\log(x^{-2}) = -2 \log(x)\) \(\log_{a} a^{n} = n\)
\(\log(bc) = \log(b) + \log(c)\) \(\log(\frac{1}{b}) = -log(b)\)

Also recall that \(n^{0} = 1, \ 1! = 1, \ 0! = 0\)
